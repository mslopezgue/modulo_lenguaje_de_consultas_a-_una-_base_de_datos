SQL
Lenguaje estructurado de consultas por el cual accedemos a hacer peticiones a las bases de datos
siempre va ir un usuario, solicitud va ir a la base de datos mediante el gestor se va devolver una respuesta al usuario.
mY SQL, ACCES, ORACLE.

El lenguaje de SQL es un lenguaje de alto nivel debido al tipo de instrucciones que utiliza textualmente.

Datos
unidad mas básica de la información, los datos se procesan y se genera la informacion ya que
por si solos no entregan nada pero tienen relevancia para el usuario. La informatica proporciona herramientas para procesar 
manipular los datos y transformarlos en información. Son significantes para quien los observa.

Informacion
conjunto de datos que poseen un significado de modo tal que reducen la oncertidumbre
y aumentan el conocimiento de quien se acerca a observarlos. Estos datos se enuentran disponibles
para uso inmediato y sirven para clarificar incertidumbres sobre determinados temas.

Bases de datos

estructura organizada de datos. se almacenan electronicamente 
de computadora. una base de datos es controlada por un sistema de 
gestión de base de datos (DBMS).
En conjunto los datos y el DBMS, junto con las aplicaciones
que estan asociados con ellos, se conocen como un SISTEMA DE BASE DE DATOS 
que a menudo se reducen a solo la base de datos.

My sqlworkbench te entrega las instrucciones, y el código SQL, es intiuitivo y amigable.

Sentencias SQL

DDL: lenguaje de definición de datos, sentencias que nos permite definir, alterar, 
modificar objetos dentro de mi base de datos.
DML: lenguaje de manipulación de datos, permite manipular, consultar, eliminar los datos o registros de las tablas.

Sentencias SQL-DML
select: recupera datos
insert añade filas
delete suprime filas de datos
Update: modifica datos existentes
Se acompañan de clausulas y operadores

Sentencias SQL-DDL
Create: nos crea distintos elementos iniciar una BD una tabla.
Drop: nos permite eliminar elelementos
Alter: modifica la estructura de la BD.

Modificar una tabla
podemos agregar nuevos campos a nuestra tabla de la siguiente manera:
alter table nombre table add_column campo tipocampo
alter table libro add column ejemplares integer;

podemos modificar el tipo de dato de un campo de nuestra tabla,
de la siguiente manera:

alter table nombre_tabla modify campo tipocampo
alter table libro modify precioVenta double;

Modificar una tabla Drop
podemos eliminar un campo de nuestra tabla de la siguiente manera:

alter table nombre_tabla drop column campo
alter table libro drop column precioVenta;

modificar una tabla Drop
podemos eliminar una tabla de la siguiente manera
drop table nombre_tabla
drop table nombre_t;
