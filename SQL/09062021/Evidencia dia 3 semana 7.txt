Uso de funciones

Función COUNT: podemos contarlos registros de nuestra tabla, se expresa de la
siguiente manera:

select count(campo) from nombre_tabla
select count(id_libro) as cantidad from Libro;

Función MAX: Podemos determinar cual es el mayor valor de un campo
en nuestros registros, se expresa de la siguiente manera:

select max(campo) from nombre_tabla
select max(edad) from tablaEmpleado;

Función MIN:  Podemos determinar cual es el mínimo valor de un campo
en nuestros registros, se expresa de la siguiente manera:

select min(campo) from nombre_tabla
select min(edad) from tablaEmpleado;

Función SUM:Podemos determinar la suma de los valores de un campo
en nuestras tablas, se expresa de la siguiente manera:

Select sum(campo) from nombre_tabla
select sum(sueldo) from tablaEmpleado;
